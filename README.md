# Minimal Hero Section

A clean and elegant hero section designed with simplicity in mind. The layout focuses on strong typography, warm earthy colors, and a centered visual element to create a premium first impression.

## ✨ Features

* Minimalistic and modern design
* Full-screen hero section
* Compact content width with reduced side margins
* Rounded-corner featured image card
* Rounded CTA button
* Strong visual hierarchy using two complementary fonts
* Responsive layout for desktop and mobile devices
* Warm, nature-inspired color palette

## 🎨 Design System

### Colors

| Purpose              | Color     |
| -------------------- | --------- |
| Background           | `#FBF5DD` |
| Secondary Typography | `#306029` |

### Typography

| Usage                        | Font      |
| ---------------------------- | --------- |
| Primary Typography           | Open Sans |

## 📐 Layout Guidelines

* Use a **narrow container width** to avoid excessive whitespace.
* Maintain generous **vertical spacing** between elements.
* Keep **horizontal margins minimal** to ensure the content feels focused.
* Position the image prominently within the hero section.
* Ensure all interactive elements have rounded corners to match the visual language.

## 🖼️ Image Guidelines

* Center-aligned within the hero section.
* Displayed inside a rounded rectangle.
* Large enough to serve as the primary visual element.
* Use high-quality portrait or lifestyle imagery.

## 🔘 CTA Button

* Rounded corners
* Clear and concise label
* Strong contrast against the background
* Positioned directly below the headline content

## 🎯 Design Goals

This hero section should communicate:

* Elegance
* Simplicity
* Warmth
* Professionalism
* Modern aesthetics

The final experience should feel like a premium portfolio, personal brand website, or modern business landing page that values clarity and refined visual storytelling.
